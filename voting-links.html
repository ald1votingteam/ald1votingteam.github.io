Supabase Setup

To save links permanently, connect your Supabase project.

Required SQL Command:

Run this in your Supabase SQL Editor:

create table voting_links ( id bigint generated by default as identity primary key, platform text, link text, type text, notes text, deadline timestamp with time zone, status text default 'active', is_new boolean default false, created_at timestamp with time zone default timezone('utc'::text, now()) );
